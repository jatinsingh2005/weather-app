# 🌦 Weather App

A Python CLI application to fetch real-time weather data using OpenWeather API.

## 🚀 Features
- 🌡 Temperature in Celsius
- 🌥 Weather description
- 💧 Humidity info
- ❌ Error handling for invalid city/API

## 🛠 Tech Used
- Python
- Requests Library
- OpenWeather API

## ▶️ How to Run
```bash
pip install -r requirements.txt
python weather.py
